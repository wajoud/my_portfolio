## ✨ Web Application & Architecture Highlights

This repository hosts Wajoud's portfolio web application built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**, deployed serverlessly on **GitHub Pages**.

<div align="center">

| Core Feature | Technical Implementation | Practical Performance Gain |
|---|---|:---:|
| ⚡ **GPU-Accelerated Marquee** | Pure CSS infinite marquee animation replacing CPU-heavy JS scroll event listeners. | **0% Scroll CPU overhead** |
| 🛡️ **Ultra-Lightweight Assets** | All project media optimized and stored in Vite's public asset pipeline. | **Instant Asset Loading** |
| 🎨 **Interactive Project Modals** | Custom modal overlay with image slider, keyboard (ESC) listener, and project metadata. | **Fluid Single Page Experience** |
| 💬 **Slide Testimonials** | Fully responsive recommendation carousel with auto-rotation (10s) and direct LinkedIn links. | **Dynamic Social Proof** |
| 📬 **Direct Mailto Gateway** | Glassmorphic contact card with direct `mailto:` integration and copy-paste fallback. | **Zero Backend Overhead** |
| ☁️ **Serverless GitHub Pages CI/CD** | Auto-built and deployed via GitHub Actions to custom domain `wajoudnoorani.com`. | **100% Uptime, $0/mo Cost** |

## 💻 Quick Start — Local Development

```bash
# 1. Clone the repository
git clone https://github.com/wajoud/my_portfolio.git
cd my_portfolio/portfolio

# 2. Install dependencies
npm install

# 3. Start development server with Hot Module Replacement (HMR)
npm run dev
```

> 🌐 Open **[http://localhost:5173](http://localhost:5173)** in your browser.

## ☁️ Production Build & Deployment

The site auto-deploys via GitHub Actions whenever changes are pushed to `main`.

To run a production build locally:

```bash
cd portfolio
npm run build
```

The compiled static assets will be generated in `portfolio/dist/`.
